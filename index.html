Current Projects

Here are a couple of the problems I am working on.

Capturing structural similarity scores

Working with Dr. Mike Mozer: Using convolutional neural networks, we are trying to train a net that can predict the structural similarity calculated using two input images. Some current code can be seen in the github repo.

Using information theory to explore paleoclimate

Working with Dr. Liz Bradley, Dr. Jim White, Dr. Joshua Garland and Dr. Tyler Jones: Using permutation entropy we are trying to analyze data collected in the water isotopic content of deep polar ice cores. The code and data are not publically available, but the last paper written by the group can be found here.

Some numerical experiments

Using monte carlo method and the stochastic process, I am learning how to use numerical analysis to solve problems that are difficult to compute analytically. Jupyter notebooks and code can be found in the github repo.
